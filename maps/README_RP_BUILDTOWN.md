# RP_BUILDTOWN Hammer Blockout

`rp_buildtown_blockout.vmf` is a first-pass Hammer VMF source map for a Garry's Mod BuildRP island based on the supplied RP_BUILDTOWN concept image.

## Created files

- `maps/rp_buildtown_blockout.vmf` — Hammer VMF source with named blockout sections, district pads, roads, core buildings, spawns, build plots, admin bunker marker, and mountain tunnel markers.
- `docs/rp_buildtown/DISTRICT_PLAN.md` — district-by-district layout plan and gameplay intent.
- `docs/rp_buildtown/COMPILE_INSTRUCTIONS.md` — Hammer setup and compile notes.
- `docs/rp_buildtown/TODO_POLISH.md` — manual polish checklist for turning the blockout into a production map.

## Map contents

The VMF includes the required island districts:

1. Spawn / Plaza
2. Shops / Downtown
3. Apartments
4. Hospital
5. Park / Recreation
6. Beach / Waterfront
7. Farmland
8. Build Zone A
9. Build Zone B
10. Industrial / Construction
11. Mountain / Forest

It also includes required roleplay landmarks: bank, police department, vehicle dealership, construction company yard, underground admin bunker shell, mountain tunnel system indicators, roads connecting all zones, player spawn points, and marked BuildRP plots.

## Recommended next step

Open the VMF in Hammer, compile with Fast VIS/RAD, verify player scale and road travel times, then follow `docs/rp_buildtown/TODO_POLISH.md` for detailed manual work.
