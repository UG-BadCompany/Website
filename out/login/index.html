Single Portal Login

One login. The right dashboard.

Clients, admins, and workers should all sign in from one secure login. After authentication, the system checks the user’s permissions and sends them to the correct dashboard or shows extra tools inside the same dashboard based on their role.

Client Permissions

View own requests, quotes, invoices, payments, messages, files, and job history.

Worker Permissions

Includes assigned jobs, job notes, checklists, status updates, materials, and completion photos.

Admin Permissions

Includes all client and worker tools plus users, quotes, scheduling, invoices, payments, reports, and settings.

Role-based routing plan

  • Client signs in → dashboard shows only their own properties, requests, quotes, and invoices.
  • Worker signs in → dashboard adds assigned jobs, task checklists, job notes, and photo uploads.
  • Admin signs in → dashboard unlocks client management, quote builder, schedules, invoices, workers, and settings.
  • If one person has multiple roles, the dashboard can show the highest allowed tools or role switcher.
